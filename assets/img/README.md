## Image Guidelines

### Formats
- **JPEG** (.jpg): For photographs and complex images
- **PNG** (.png): For logos, icons, and images with transparency
- **SVG** (.svg): For simple graphics, icons, and scalable logos
- **WebP** (.webp): For modern browsers (optional optimization)

### Size Recommendations
- **Logo**: 200x80px (2x for retina: 400x160px)
- **Hero Images**: 1200x600px minimum
- **Feature Images**: 400x300px
- **Team Photos**: 300x300px (square format)
- **Product Images**: 500x500px (square format)
- **Partner Logos**: 200x100px

### Optimization
- Compress images to reduce file size
- Use appropriate quality settings (80-90% for JPEG)
- Consider using tools like TinyPNG or ImageOptim
- Provide 2x versions for retina displays when possible

### Naming Convention
- Use lowercase letters
- Separate words with hyphens
- Be descriptive but concise
- Include size suffix when applicable (e.g., logo-2x.png)

### Accessibility
- Always provide meaningful alt text in HTML
- Use descriptive filenames
- Ensure sufficient contrast for text overlays
